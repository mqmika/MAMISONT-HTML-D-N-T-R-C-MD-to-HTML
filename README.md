# MAMISONT-HTML-D-N-T-R-C-MD-to-HTML
Markdown dosyasını seç Dönüştürmek istediğin .md dosyasını uygulamaya yükle.  Stil ayarlarını yap Gelişmiş ayarlar menüsünden renkler, fontlar ve mouse imleci dahil olmak üzere tüm estetik seçenekleri belirle.  HTML olarak kaydet “Dönüştür ve Kaydet” butonuna tıkla. Uygulama, tüm ayarlarınla birlikte HTML çıktısını otomatik olarak indirir.
